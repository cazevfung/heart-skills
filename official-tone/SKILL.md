# Official Tone Skill

本 skill 是游戏官方内容创作的**风格参考库**，回答"这个游戏官方说话是什么感觉"。

## 使用方法

1. **确定游戏** - 用户提及的游戏名称
2. **查询 tone sheet** - 阅读 `references/[游戏].md`
3. **理解气质** - 把握语调、视角、视觉感觉
4. **应用到创作** - 撰写符合该游戏官方风格的内容

## 核心原则（跨游戏通用）

### 原则1：官方是"谁"

每个游戏官方都有一个"人格"。问自己：
- 这个官方是玩家的什么人？（朋友/导师/陪伴者/观察者）
- 这个官方什么性格？（活泼/沉稳/神秘/温暖）
- 这个官方和玩家关系如何？（亲密/尊敬/平等/疏离）

### 原则2：语调即品牌

玩家能从一句话判断"这是XX游戏的官方"。

关键元素：
- 常用词汇与句式
- 句子长度与节奏
- 情绪温度（热情/克制/幽默/严肃）
- 人称使用（我/你/我们/玩家）

### 原则3：场景适配

同一游戏在不同场景下语调微调：

| 场景 | 调整方向 |
|------|----------|
| 版本前瞻 | 最完整表达，可带情绪高潮 |
| 更新公告 | 清晰优先，情绪收敛 |
| 社媒日常 | 轻松随意，可玩梗 |
| 客服/邮件 | 专业友好，解决问题导向 |
| 危机回应 | 真诚直接，不回避 |

### 原则4：反模式识别

知道"什么感觉是这个游戏官方不会有的"同样重要。

每个 tone sheet 包含反模式，帮助避免"出戏"。

## 新增游戏 tone sheet 指南

当需要为新游戏创建 tone sheet 时，包含以下模块：

```markdown
# [游戏名] Official Tone

## 气质定义
这个游戏官方是什么"性格"？和玩家是什么关系？

## 语调特征
- 常用句式
- 词汇偏好
- 节奏特点
- 情绪温度

## 视觉感觉
画面、排版、色彩的参考锚点

## 叙事视角
官方与玩家的关系定位

## 反模式
这个游戏官方不会有的感觉

## 场景适配
不同内容类型的微调

## 参考锚点
具体的参考对象（其他游戏、品牌、媒体等）
```
