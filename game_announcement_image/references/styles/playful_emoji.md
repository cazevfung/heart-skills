# 活泼表情 — 公告图视觉风格（Playful + Emoji）

用于**图像生成**时的视觉与构图指引，与文案风格「活泼表情」对应。适合活动、福利、节日类公告。

## 色调与氛围

- 明亮、有活力：高饱和但不刺眼，可多色搭配（如活动主题色 + 互补色）
- 渐变、光效、轻微动感均可；节日可应景（如红金、圣诞、春节）
- 整体「开心、想点进去」的感觉
- 避免：灰暗、沉闷、纯正式风

## 构图与版式

- 可更自由：主标题突出，副信息用图标或色块区分
- 留白与信息密度平衡；可适当「满」一点，体现活动感
- 若有角色/吉祥物，可更大、更显眼，与活动主题呼应
- 适合圆角、波浪、气泡等轻松形状

## 字体与文字感

- 标题：醒目、可带圆体或活泼无衬线，可配合描边/阴影增强存在感
- 正文：清晰但偏轻松；可配合小图标、emoji 位置预留
- 整体感觉：像活动海报、福利预告，而非正式公文
- 可配合：星星、礼包、时钟等轻量图形元素

## 元素与禁忌

- **宜**：礼包、星星、彩带、角色 Q 版、活动主题 icon、适度高光
- **忌**：纯文字墙、冷色主调、严肃边框、恐吓式视觉
- 与「活泼表情」文案一致：像运营组发的活动图，有号召力

## 生成 prompt 建议（供 LLM/API 使用）

- 可包含：vibrant game event announcement, playful style, festive colors, gift or reward theme, friendly mascot optional, clean but fun layout, eye-catching title
- 避免：corporate, serious, dark, minimal only, no decoration
