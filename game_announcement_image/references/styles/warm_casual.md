# 温暖口语 — 公告图视觉风格（Warm & Casual）

用于**图像生成**时的视觉与构图指引，与文案风格「温暖口语」对应。适合社区向、治愈系、生活模拟类游戏。

## 色调与氛围

- 暖色为主：米白、浅杏、淡黄、柔和橙、低饱和的绿/蓝点缀
- 整体明亮但不刺眼，避免高对比、冷灰或金属感
- 可带轻微渐变或柔和光晕，营造「温暖、安心」感
- 避免：纯黑底、荧光色、赛博/科技冷色

## 构图与版式

- 留白适中，信息不堆满；标题区与正文区可分明
- 主视觉可偏左或居中，留出呼吸感
- 若有角色/吉祥物，建议偏小、不抢戏，与文字平衡
- 适合圆角矩形、柔和分块，少用锐利几何

## 字体与文字感

- 标题：圆润、易读的无衬线或轻度圆体，字号突出
- 正文：清晰可读，行距略松
- 整体感觉：亲切、手写感或柔和印刷感均可，避免冷硬黑体或过于正式的衬线
- 可配合少量手绘感装饰（小图标、简单插画元素）

## 元素与禁忌

- **宜**：小图标、简单插画、植物/自然元素、柔和几何、游戏角色 Q 版
- **忌**：严肃公文风、大量边框与表格感、军事/硬核符号、过于复杂的纹理
- 与「温暖口语」文案一致：像朋友发的公告图，而非政府通知

## 生成 prompt 建议（供 LLM/API 使用）

- 可包含：warm color palette, soft lighting, casual friendly game announcement, cozy community style, minimal clean layout, rounded shapes, gentle gradient background
- 避免：corporate, formal, dark, high contrast, tech style
