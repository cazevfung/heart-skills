# 正式简洁 — 公告图视觉风格（Official Formal）

用于**图像生成**时的视觉与构图指引，与文案风格「正式简洁」对应。适合版本说明、维护公告、海外或偏硬核向游戏。

## 色调与氛围

- 偏中性、专业：深蓝、灰、白、或游戏主色但降低饱和度
- 对比清晰，信息层级分明；可保留少量品牌色强调
- 整体干净、偏「官网/公告栏」感，不花哨
- 避免：过于活泼的渐变、大量装饰、卡通化

## 构图与版式

- 版式规整：标题区、正文区、落款区可明确分区
- 对齐工整，留白有节奏；适合网格化布局
- 主视觉可为 logo、版本号、或简洁图形，不喧宾夺主
- 适合直角或轻微圆角，偏「产品/品牌」感

## 字体与文字感

- 标题：无衬线或现代衬线，字重足够，易读
- 正文：清晰、行距适中，便于扫读
- 整体感觉：专业、可信，避免手写体或过于活泼的字体
- 可配合简单图标（日历、时间、勾选等）增强信息传达

## 元素与禁忌

- **宜**：logo、版本号、简洁图标、分隔线、清晰区块
- **忌**：大量 emoji、卡通角色抢戏、杂乱装饰、过于娱乐化的配色
- 与「正式简洁」文案一致：像官方公告图，信息优先

## 生成 prompt 建议（供 LLM/API 使用）

- 可包含：professional game announcement, clean layout, official style, minimal design, clear typography, corporate color palette, structured sections
- 避免：casual, playful, cartoon, decorative, cluttered
