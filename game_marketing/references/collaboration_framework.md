# 联动认知体系

本文件供 `game_marketing` skill 在产出地区营销方案时使用。用于在「渠道策略」「本地化重点」或单独「联动规划」中，系统化考虑联动（跨界合作、IP 联动、品牌联名等）的类型、选品逻辑、与节奏的结合及风险控制。

---

## 一、联动的定义与类型

**联动**：游戏与外部 IP、品牌、艺人、文化符号或线下场景的合作，通过内容/权益/体验的融合实现破圈、拉新、品牌背书或长线口碑。

| 类型 | 说明 | 典型形态 | 主要效果 |
|------|------|----------|----------|
| **IP/动漫联动** | 与动漫、影视、小说等 IP 角色/剧情植入 | 限定角色、剧情副本、皮肤、剧情番外 | 拉新 IP 粉丝、双厨狂喜、内容话题 |
| **品牌联名** | 与消费品牌、快消、时尚等推出联名商品/活动 | 联名款、限定包装、线下快闪、礼包 | 破圈、品牌调性、线下曝光 |
| **音乐/艺人联动** | 主题曲、演唱会、艺人代言/植入 | 主题曲、MV、虚拟演唱会、代言人 PV | 情感联结、泛用户触达、话题 |
| **文旅/文化联动** | 与地域文化、景区、非遗、博物馆合作 | 取景地、文化剧情、线下地标、非遗元素 | 文化背书、长线 IP、本地化亲和 |
| **节日/节点联动** | 与节日、赛事、大促节点绑定 | 限定活动、皮肤、剧情、福利 | 档期集中、心智占位 |
| **KOL/达人联动** | 与多圈层 KOL、主播、UP 主合作 | 试玩、二创、直播、定制内容 | 信任背书、圈层渗透、UGC |

同一项目中可组合多种类型（如「IP 联动 + 音乐主题曲 + 文旅取景地」）；需与目标地区、用户画像和档期一致。

---

## 二、联动选品与选伙伴的考量

**联动须优先评估用户群体匹配**——本品目标用户（来自 STP/用户画像）与联动方受众的重叠度与一致性；不匹配易导致转化差或口碑反弹。

在方案中建议联动方向或具体伙伴时，可按以下维度评估，并在无数据时注明「待定」或「需商务补充」。

| 维度 | 要点 |
|------|------|
| **用户群体匹配（用户契合）** | 联动方受众与本品核心/次核心用户重叠度；与本品该地区用户画像（核心用户、兴趣标签、触媒习惯）的匹配度；是否有助于触达目标画像。 |
| **内容/调性契合** | 世界观、美术风格、价值观是否可融合；玩家是否易接受「不违和」。 |
| **档期与节奏** | 与上线 D-Day、版本节点、竞品大动作的错位或借势；是否有足够筹备期。 |
| **资源与成本** | 版权/授权、制作、渠道投放的大致量级；与本地化预算、KOL 预算的占比关系。 |
| **风险** | 联动方舆情、政治/文化敏感、版权纠纷、玩家预期过高导致「货不对板」；需在方案中做简要风险提示。 |

**输出建议**：在分地区方案中须给出「建议联动类型」「优先方向（如 IP/音乐/文旅）」「档期建议（如 D+30 首版本联动）」「风险提示（1–2 条）」；**须说明与目标用户画像的匹配理由（1 句）**。具体伙伴名可写「待定」或示例。

## 三、联动与渠道、里程碑、本地化的结合

- **渠道策略**：联动内容作为「内容形式」的一部分（如联动 PV、联动直播、联名商品首发）；主力渠道（TikTok/YouTube/本地渠道）可配合联动话题与素材。
- **里程碑**：D-30 可安排联动预告/官宣；D-Day 可绑定联动福利或限定；D+30 可做首版本联动活动，与留存、UGC 结合。
- **本地化重点**：不同地区优先的联动类型可不同（如日本偏动漫/IP、东南亚偏本地 KOL 与支付/渠道合作）；文化适配时需考虑联动方的当地认知与合规。

---

## 四、常见风险与成功要素

**风险**：联动方负面舆情波及本品；文化/政治敏感（尤其多地区发行）；承诺内容未兑现或品质低于预期；版权/授权纠纷。

**成功要素**：用户与内容双契合；档期与资源匹配；预期管理（宣传不过度承诺）；舆情与合规预检；长线可复用（如文旅、音乐资产沉淀）。

在方案中若给出联动建议，应至少包含 1 条风险或前提条件（如「需提前确认授权与档期」），避免读起来像已确定执行项。
