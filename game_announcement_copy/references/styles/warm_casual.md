# 温暖口语（Warm & Casual）

## 一句话描述

亲切、像朋友说话，带一点治愈感，适合社区向、生活模拟类游戏。

## 语气 / 口吻

- 亲切、温暖，不冷冰冰
- 口语化但不随便，避免过于正式或公文腔
- 可适当拟人化（如用「我们」代表运营/开发，把玩家当邻居、居民）
- 适度情绪词：开心、期待、感谢

## 称呼与署名

- 称呼玩家：亲爱的居民 / 小镇居民 / 各位小伙伴 / 亲爱的玩家（择一与游戏设定一致）
- 自称：我们 / 小镇居委会 / 运营小分队（与游戏官方人设一致）
- 落款：可用简短一句 + 团队名，如「—— 小镇居委会」「我们在小镇等你～」

## 用词偏好与禁忌

- **偏好**：一起、期待、感谢、小小的、马上、很快、稍等、放心
- **避免**：兹、鉴于、特此通知、勒令、严禁、必须、否则不予
- **避免**：过长从句、法律/公文句式、纯英文缩写无解释（若面向中文玩家）

## 结构建议

- **开头**：简短问候或直接点题（如「大家好啊」「有一件事要和大家说」）。
- **正文**：按要点分段，每段一两句即可；重要时间、补偿用加粗或单独一行突出。
- **结尾**：一句感谢或期待（如「感谢大家的理解」「我们不见不散」），再落款。

## 公告类型差异

- **维护公告**：先说明「我们要做一点小升级/维护」，再给时间与范围，最后说明补偿与感谢理解。
- **活动公告**：可更活泼，突出「有什么好玩/福利」，号召参与。
- **版本/更新**：可强调「新内容来了」，用短句列亮点，结尾邀请体验与反馈。

## 示例片段（脱敏）

- 开头示例：「亲爱的居民们，有一则小通知要和大家说～」
- 维护示例：「我们将在 X 月 X 日 X:00–X:00 进行短暂维护，期间无法登录。维护结束后会为大家奉上小小心意，感谢理解！」
- 结尾示例：「—— 小镇居委会，我们在小镇等你。」
