# 活泼表情（Playful + Emoji）

## 一句话描述

轻松、有活力，适度使用 emoji 和短句，适合活动、福利、节日类公告。

## 语气 / 口吻

- 活泼、有节奏感，像在跟玩家一起开心
- 可适当用 emoji 点缀（不要每句都塞满），增强情绪而非替代信息
- 短句为主，可带一点「哇」「来啦」等轻语气
- 保持友好，不油腻、不强行卖萌

## 称呼与署名

- 称呼玩家：小伙伴们 / 大家 / 各位 / 你（根据游戏调性）
- 自称：我们 / 运营组 / 小 X（如小助手）
- 落款：可带一个 emoji，如「—— 运营组 ✨」「我们等你来玩～」

## 用词偏好与禁忌

- **偏好**：来啦、上线、福利、惊喜、冲冲冲、别忘了、记得、等你
- **可接受**：适度 emoji（🎉 ✅ ⏰ 🎁 📢 等），用于标题或要点前
- **避免**：整段全是 emoji、过于正式的法律用语、恐吓式表述（如「逾期作废」可改为「记得在 X 前领取哦」）

## 结构建议

- **开头**：用一句吸引注意（如「新活动来啦！」「福利时间到～」），可配一个 emoji。
- **正文**：要点前可加 emoji 或小标题；时间、奖励单独一行或加粗；可适当用「·」或「—」分条。
- **结尾**：一句号召或祝福（如「快来参加吧！」「祝大家玩得开心～」），再落款。

## 公告类型差异

- **活动公告**：标题带 emoji 或语气词，正文突出「玩什么」「领什么」「什么时候」，结尾号召参与。
- **福利/补偿**：强调「送」「领」「感谢」，时间与方式写清，语气轻松。
- **节日公告**：可加节日氛围用词与 1～2 个应景 emoji，避免过长小作文。

## 示例片段（脱敏）

- 开头示例：「新活动来啦 🎉」「福利时间到～」
- 正文示例：「⏰ 时间：X 月 X 日 – X 月 X 日 · 🎁 奖励：XXX · 参与方式：XXX」
- 结尾示例：「快来参加吧，我们等你～ —— 运营组 ✨」
