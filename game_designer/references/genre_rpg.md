# RPG（含 ARPG / JRPG）

**代表作：** 崩坏3、仙剑、最终幻想、黑神话·悟空

- **核心玩法循环：** 剧情/任务驱动 → 战斗与成长 → 获得装备/经验/技能 → 提升战力后推进剧情或挑战更高难度，形成「叙事—战斗—成长—再叙事」。
- **关键设计系统：** 角色/职业与技能体系、数值成长曲线、装备与词条、剧情与分支、副本与 BOSS 设计、难度梯度。
- **玩家核心动机：** 故事与角色认同、成长感与数值反馈、战斗手感与策略深度。
- **常见设计陷阱：** 数值膨胀导致旧内容废弛；剧情与玩法脱节；成长曲线过陡或过平；重复刷本缺乏变化导致疲劳。
- **与商业化/留存的交叉点：** 付费常与抽卡、装备、通行证结合；留存依赖剧情更新与有意义的成长目标；需注意付费与公平感（尤其是 PVP/共斗向）。
