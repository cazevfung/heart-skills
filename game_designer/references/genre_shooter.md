# 射击（FPS / TPS）与战术竞技

**代表作：** 和平精英、CS2、无畏契约、APEX

- **核心玩法循环：** 单局匹配 → 跳伞/选点—搜刮—交战—缩圈/目标 → 存活或达成目标 → 结算与排位/通行证进度 → 再开一局。
- **关键设计系统：** 枪械与手感、地图与圈/目标、匹配与排位、通行证与外观、反外挂与公平。
- **玩家核心动机：** 竞技公平、操作与枪感、生存与胜利快感。
- **常见设计陷阱：** 外挂与公平性；付费影响数值会直接崩盘；地图与节奏单一导致疲劳。
- **与商业化/留存的交叉点：** 外观与通行证为主，严禁 Pay to Win；留存依赖对局质量与赛季/通行证目标。
