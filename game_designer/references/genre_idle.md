# 放置 / 挂机

**代表作：** 放置奇兵、咸鱼之王、剑与远征、AFK Journey

- **核心玩法循环：** 挂机产出资源 → 消耗资源升级/抽卡/养成 → 推图或爬塔验证 → 继续挂机，形成「挂机—养成—验证—再挂机」。
- **关键设计系统：** 挂机产出与离线收益、资源与数值曲线、英雄/卡牌与阵容、推图/爬塔/活动、每日与长线目标。
- **玩家核心动机：** 低负担成长感、数值反馈、收集与阵容搭配。
- **常见设计陷阱：** 付费与免费进度差距过大；中后期仅剩数值堆砌缺乏策略；内容消耗后仅剩重复挂机。
- **与商业化/留存的交叉点：** 付费常加速挂机或抽卡；需保留免费玩家可玩空间；留存依赖阶段性目标与轻度策略深度。
