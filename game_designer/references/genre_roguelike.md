# Roguelike / Roguelite

**代表作：** 以撒的结合、枪火重生、哈迪斯、吸血鬼幸存者

- **核心玩法循环：** 单局随机地图/道具/事件 → 决策与战斗 → 死亡或通关 → 解锁永久成长或新内容 → 再开一局，形成「单局—死亡/通关—永久成长—再局」。
- **关键设计系统：** 单局随机与种子、永久解锁与元进度、难度与可选挑战、Build 与决策深度。
- **玩家核心动机：** 单局变化与重玩价值、Build 与运气博弈、挑战与解锁成就感。
- **常见设计陷阱：** 随机导致挫败感过强；永久成长过强削弱 Rogue 味；内容消耗后重复感明显。
- **与商业化/留存的交叉点：** 多为买断或内购 DLC；若含内购需避免破坏单局公平与重玩价值；留存依赖单局多样性与解锁目标。
