# 二次元（风格/受众维度）

**代表作：** 崩坏星穹铁道、碧蓝航线、原神、明日方舟

- **核心玩法循环：** 视底层玩法而定（可为卡牌、RPG、塔防等），叠加「角色—剧情—视觉—社区」的情感循环：获取/养成角色 → 消费剧情与二创 → 认同与归属。
- **关键设计系统：** 角色人设与立绘/建模、剧情与世界观、声优与音乐、活动与限定、社区与二创氛围。
- **玩家核心动机：** 角色情感、审美与认同、收藏与「厨力」、社区归属。
- **常见设计陷阱：** 角色强度或剧情争议引发炎上；和谐与本地化引发不满；过度商业化伤害情感信任。
- **与商业化/留存的交叉点：** 抽卡与角色情感强绑定；留存依赖角色与剧情质量及社区正向氛围；公告与文案需贴合二次元语境与敏感点。
