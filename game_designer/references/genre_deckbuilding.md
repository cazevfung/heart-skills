# 卡牌构筑（Deck-building）

**代表作：** 杀戮尖塔、怪物火车、月圆之夜

- **核心玩法循环：** 用当前牌组战斗 → 获得新牌或删牌/升级 → 构筑更强牌组 → 挑战下一层/ Boss，形成「战斗—获得—构筑—再战斗」。
- **关键设计系统：** 卡牌与流派、随机获得与删牌、关卡与 Boss、单局与可选永久进度。
- **玩家核心动机：** 构筑与策略深度、单局变化与重玩、挑战与突破。
- **常见设计陷阱：** 平衡导致套路单一；随机过重导致不可控挫败；长线内容不足。
- **与商业化/留存的交叉点：** 多为买断或扩展包；若抽卡化需谨慎不破坏构筑逻辑；留存依赖新内容与难度梯度。
