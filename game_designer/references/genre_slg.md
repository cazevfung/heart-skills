# SLG / 策略

**代表作：** 三国志·战略版、率土之滨、万国觉醒、文明

- **核心玩法循环：** 占地/城建/养兵 → 联盟协作与对抗 → 赛季或服务器目标（征服/排名）→ 新赛季重置或迭代，形成「发展—联盟—对抗—赛季」。
- **关键设计系统：** 地图与势力、资源与城建、兵种与克制、联盟与外交、赛季规则与重置、付费与加速。
- **玩家核心动机：** 策略深度、联盟归属与霸主地位、长线谋略与局势感。
- **常见设计陷阱：** 顶级付费碾压导致免费玩家流失、服崩；单赛季过长或过短；联盟管理负担过重；付费加速破坏策略公平感。
- **与商业化/留存的交叉点：** 付费差距大，需保证免费玩家生存与参与感；留存依赖联盟粘性与赛季目标；健康度需评估「付费上限」与服务器生态平衡。
