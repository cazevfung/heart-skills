# 开放世界

**代表作：** 原神、黑神话·悟空、塞尔达传说、幻兽帕鲁

- **核心玩法循环：** 探索地图与据点 → 发现任务、资源、敌人与谜题 → 完成挑战获得成长与奖励 → 解锁新区域或能力 → 继续探索，形成「探索—发现—解决—再探索」。
- **关键设计系统：** 大世界结构与引导（地标、视野、任务线）、移动与交通、资源分布与采集、动态事件与世界反应、主线/支线/世界任务分层、地图与进度可视化。
- **玩家核心动机：** 自由感与发现感、沉浸式世界、按自身节奏推进。
- **常见设计陷阱：** 地图空洞、重复感强；引导过强削弱探索感或过弱导致迷路与弃坑；性能与加载影响流畅探索；内容消耗速度与产能不匹配导致长草。
- **与商业化/留存的交叉点：** 多为内购或买断；若含 Gacha/养成，需避免「开放世界」被付费墙割裂；留存依赖持续的地图/剧情/活动更新与探索动机维持。
