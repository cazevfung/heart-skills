# MMO / MMORPG

**代表作：** 梦幻西游、天涯明月刀、逆水寒、魔兽世界

- **核心玩法循环：** 角色成长与装备提升 → 副本/团本/PVP/社交 → 获得奖励与地位 → 持续成长与社交归属，形成「成长—参与—回报—归属」。
- **关键设计系统：** 职业与天赋、装备与掉落、副本与团本、公会/帮派与社交、经济系统（交易、拍卖）、赛季或资料片节奏。
- **玩家核心动机：** 成长感、社交归属、服务器内地位与认同。
- **常见设计陷阱：** 大 R 与免费玩家差距过大导致生态崩溃；内容消耗后长草；社交绑架或公会压力导致负面体验。
- **与商业化/留存的交叉点：** 订阅/月卡与道具并存；大 R 占比高但需保护免费玩家生存空间；留存强依赖社交与持续内容更新。
