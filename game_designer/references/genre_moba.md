# MOBA

**代表作：** 王者荣耀、英雄联盟、决战平安京

- **核心玩法循环：** 单局匹配 → 选人/禁选 → 对线/团战/推塔 → 结算与段位/战力变化 → 再开一局，形成「对局—结算—再对局」。
- **关键设计系统：** 英雄设计与平衡、地图与资源点、经济与装备、匹配与排位、赛季与版本节奏。
- **玩家核心动机：** 竞技公平、操作与策略成就感、社交与段位认同。
- **常见设计陷阱：** 平衡崩坏导致部分英雄或套路统治；付费影响公平会直接崩盘；消极行为与匹配体验影响留存。
- **与商业化/留存的交叉点：** 以外观付费为主，付费不得影响数值公平；留存强依赖对局体验与赛季目标；健康度重点在「付费是选择而非强迫」。
