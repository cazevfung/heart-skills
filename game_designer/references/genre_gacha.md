# 卡牌 / Gacha

**代表作：** 原神、明日方舟、蔚蓝档案、崩坏星穹铁道

- **核心玩法循环：** 获取角色/卡牌 → 养成（等级、突破、技能、装备）→ 挑战关卡/副本/活动 → 获得资源与抽卡货币 → 再次获取与养成，形成「收集—养成—验证」循环。
- **关键设计系统：** 抽卡与概率（含保底、UP、井）、角色/卡牌强度与稀有度分层、资源经济（体力、货币、突破材料）、关卡难度曲线与数值门、活动与限时卡池节奏。
- **玩家核心动机：** 角色情感认同与收藏欲、强度验证与攻略成就感、剧情与世界观沉浸。
- **常见设计陷阱：** 概率不透明或保底过深引发信任危机；老角色/旧卡快速贬值（背刺）导致付费资产受损；活动与卡池节奏过密导致疲劳与付费压力；核心玩法单一导致长线仅剩抽卡驱动。
- **与商业化/留存的交叉点：** 付费天花板高，首充与月卡设计关键；留存依赖内容更新（新角色、新剧情、新活动）与情感绑定；健康度三问中「历史资产是否被保护」在本品类尤为敏感。
