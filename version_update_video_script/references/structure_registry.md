# 结构注册表

所有可用的视频结构在此表定义。新增结构 = 在本表加一行；若需段落定义与时长建议，可新建 `references/structure_<structure_id>.md`。

| structure_id | 名称 | 触发关键词 | 段落顺序 | 详细说明（可选） |
|-------------|------|------------|----------|------------------|
| standard | 标准版 | 标准、常规、完整 | hook → 版本概览 → 按类别逐条（新角色/平衡/系统/Bug）→ 总结/CTA | [structure_standard.md](structure_standard.md) |
| fast_cuts | 快剪版 | 快剪、短平快、精简 | 3 秒钩子 → 要点罗列（每条一两句）→ CTA | [structure_fast_cuts.md](structure_fast_cuts.md) |
| story_driven | 故事线 | 故事、叙事、悬念 | 悬念开场 → 问题/背景 → 更新如何解决 → 收束 | — |
| sandwich_wave | 三明治波浪式 | 动森风、治愈系、生活模拟、Direct风格、三明治结构 | 开场钩子 → 免费内容波浪堆叠 → 情感转折 → 付费内容深度展示 → 收尾号召 | [structure_sandwich_wave.md](structure_sandwich_wave.md) |

每段在生成时对应稿子中的一段；详细说明中可规定每段的字数/句数/时长建议、是否带时间戳标记。
