# 风格：正式官方 (formal)

适用于官方频道、品牌向、需要稳重感的版本更新视频稿。

## 口吻要求

- **人称**：以「我们」指代官方/开发团队，或以中性第三人称陈述；避免「我」个人化表达。
- **语气**：平稳、信息密度高，不煽动情绪；可适度强调「为玩家带来」「本次更新旨在」等品牌话术。
- **禁忌**：不用网络梗、流行语、夸张吐槽；不调侃平衡或 Bug，仅客观陈述改动。

## 写作要求

- 用词规范：优先使用 patch notes 原文中的术语（英雄名、技能名、数值），避免口语化简称（除非该简称在社区已等同于正式名称）。
- 句式：以短句为主，每条更新点一句说完；可适当使用「本次更新包含」「此外」「同时」等衔接词。
- 结构清晰：按「新内容 → 平衡调整 → 系统优化 → Bug 修复」等逻辑分组，便于观众快速扫读。

## 示例片段（中文）

> 各位玩家好，这里是【游戏名】运营团队。本次版本 **v1.2.0** 已于 X 月 X 日上线，我们为大家整理了本期更新要点。
>
> **新内容方面**，我们新增了英雄【XXX】，定位为……；同时开放了赛季「XXX」及相关活动。
>
> **平衡性调整**，我们对以下英雄进行了数值与机制优化：……

## 示例片段（英文）

> Hello everyone, this is the [Game Name] team. Version **1.2.0** is now live as of [date]. Here’s a concise overview of this update.
>
> **New content:** We’ve added hero [Name], designed as … ; Season "[Name]" and related events are now available.
>
> **Balance changes:** We’ve adjusted the following heroes: …
