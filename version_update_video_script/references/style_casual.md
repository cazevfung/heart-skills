# 风格：轻松易懂 (casual)

适用于 UP 主/主播口播、面向普通玩家的版本解读，像朋友在跟你讲「这版本改了啥」。

## 口吻要求

- **人称**：多用「你」拉近距离；口播者可用「我们」表示和观众同一立场（「我们玩家终于等到了」）。
- **语气**：轻松、有起伏，可带一点小情绪（「这波加强舒服了」「这个修复早该来了」）；避免过于正式或冗长。
- **适度玩梗**：可使用该游戏社区内常见梗或通用网络用语，但不过度，以免影响信息传递。

## 写作要求

- 口语化：写出来的句子要「能直接念」，避免书面长句；可适当用「然后」「再说」「另外」等口头衔接。
- 信息提炼：不必逐字照念 patch notes，用「人话」概括（例如「某某技能 CD 短了，更灵活了」而不是只报数值）。
- 节奏：重要更新点可稍展开一句；次要的 Bug 修复可一笔带过（「还有一些小修小补，就不细说了」）。

## 示例片段（中文）

> 大家好，新版本来了！这期咱们快速过一遍 v1.2 都改了啥。
>
> 首先新英雄【XXX】上线了，玩起来有点像……，喜欢这类定位的可以试试。然后新赛季也开了，奖励和之前差不多，肝就完事。
>
> 平衡这块，【某英雄】终于被砍了一刀，之前太离谱了；【另一英雄】小加强，可以多掏出来玩玩。别的就是一堆 Bug 修了，体验会稳不少。

## 示例片段（英文）

> Hey everyone, new patch is here. Let’s run through what’s in v1.2.
>
> First up, new hero [Name] is in — plays a bit like … so if you’re into that, give it a shot. New season’s live too, same grind as before.
>
> Balance-wise, [Hero] got the nerf we expected; [Other] got a small buff, worth trying again. Rest is mostly bug fixes, should feel a bit smoother.
