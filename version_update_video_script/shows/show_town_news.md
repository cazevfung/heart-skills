# 小镇新鲜事（心动小镇专属栏目）

## 栏目定位

轻量级运营向内容栏目，由官方角色（喵喵/巴布）主持，聊聊近期更新小看点 + 回应玩家高频问题（喵答Q&A）。

**风格关键词**：亲切、轻松、有对话感、角色代入

## 固定结构

| 顺序 | 段落 | 说明 | 节奏建议 |
|------|------|------|----------|
| 1 | 开场白 | 角色问候 + 本期主题预告 | 热情、短句、破折号停顿 |
| 2 | 看点速览 | 1-3 个近期更新小看点 | 信息密度中等，重点突出 |
| 3 | 喵答 Q&A | 3-5 个玩家高频问题及回答 | 问答分明，答案保护原意 |
| 4 | 结尾 | 下期预告 + CTA + 角色告别 | 温暖、期待感、固定句式 |

## 各段落写作约束

### 1. 开场白

**必须包含**：
- 角色名（从语料库读取，默认"喵喵"）
- 玩家称呼（"发展家们"）
- 本期核心看点预告（1 句话）

**句式特点**：
- 多用短句、断句
- 用"——"制造停顿和强调
- 语气词："呀~""呢~""哦"

**示例**：
```
巴布，发展家们好呀~
大事来了！就在4月25日——
心动小镇首届 "河狸杯" 建造节 就要到来啦！
今天呢，喵喵会带大家一起速看建造节的主要看点，
还有你说喵答问答环节，和发展家们聊聊近期关注的内容和优化~
马上开始今天的内容吧！
```

### 2. 看点速览

**处理逻辑**：
- 用户提供 talking points（可能含【重点】标记）
- Skill 识别重点，确保在稿子里优先呈现
- 非重点内容可合并或简化

**写作要点**：
- 用"——而且""对了""重点是"等衔接词强调卖点
- 数字/日期用口语化表达（"4月25日"而非"2025-04-25"）
- 适度展开一句解释"为什么重要"

**【重点】标记规则**：
- 用户输入中含 `【重点】` `【必讲】` `【核心】` 的内容
- 必须在稿子里完整保留
- 可适当加粗或前置

### 3. 喵答 Q&A

**输入格式**（用户提供）：
```
Q: 订购券占了很多背包容量，没有被盖章的订购券可以在背包内合并堆叠摆放吗？
A: 有（延展内容待补充）

Q: 订购机返场降价上线了吗？会否返还差价？
A: 4月16日降价已经上线啦，返还差价机制待补充
```

**处理规则**：
1. **问题**：原样保留，可适当精简冗余词
2. **答案**：
   - 核心信息 95% 保留（只改标点/衔接词）
   - 敏感表述（机制/数值/时间）不得改动
   - 占位符 `(待补充)` 粗粒度标记，保留原样
3. **格式**：
   ```
   接下来是今天的喵答Q&A环节~
   我们挑了几个大家最近问得比较多的问题聊一聊——

   Q: [问题]
   A: [答案，含占位符]

   Q: [问题]
   A: [答案]
   ```

**禁止行为**：
- ❌ 改写答案的核心意思
- ❌ 删除占位符标记
- ❌ 添加未确认的机制细节

### 4. 结尾

**必须包含**：
- 下期预告（"下个月还有一大波更新"）
- CTA（"记得4月25日来报名"）
- 角色告别（从语料库读取固定句式）

**示例**：
```
好啦，今天的分享就到这儿！
下个月还有一大波更新内容在等着大家，可以小小期待一下～
记得4月25日来报名 "河狸杯" 建造节哦！
提前祝各位发展家们假期快乐，拜拜～👋✨
```

## 语料库依赖

本栏目依赖 `config/town_corpus.json` 中的以下字段：

| 字段 | 用途 | 示例 |
|------|------|------|
| `host_characters` | 开场白角色名 | "喵喵" |
| `player_term` | 玩家称呼 | "发展家" |
| `fixed_events` | 活动名标准化 | "河狸杯" → "心动小镇首届河狸杯建造节" |
| `common_phrases.greeting` | 开场问候 | "发展家们好呀~" |
| `common_phrases.transition` | 转场句式 | "接下来是今天的喵答Q&A环节~" |
| `common_phrases.ending` | 结尾告别 | "提前祝各位发展家们假期快乐，拜拜～👋✨" |

## 与通用参数的映射

当用户触发 `town_news` 栏目时，自动映射：

| 参数 | 固定值 | 说明 |
|------|--------|------|
| `style` | `casual` | 轻松易懂口吻 |
| `structure` | `interactive_show` | 互动型结构（含Q&A） |
| `rhythm` | `conversational` | 对话感节奏（短句、停顿） |
| `interaction` | `heavy` | 重互动（角色代入+问答） |
| `draft_mode` | `collaborative` | 协作模式（支持占位符） |
