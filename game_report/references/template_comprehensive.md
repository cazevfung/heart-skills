# 综合分析报告模板（参考）

本模板作为**参考素材**，在撰写综合分析类报告时可查阅。报告结构不由模板决定，而由 Phase 3.0 自我追问的「核心发现清单」决定。

**使用方式：**
- 先完成自我追问五层框架（表象→矛盾→归因→推演→机会）
- 根据追问结果决定需要哪些 section、什么顺序
- 不确定某 section 如何撰写时，参考本模板的写作方式

**通用撰写原则：**
- 开篇结论必须为单独一段 8–15 句，仅文字，不含表格或列表
- 每个一级 section 必须包含**至少 2–3 段连贯的深度解读段落**（每段 3–5 句）
- **禁止在报告中使用任何 emoji**；等级、程度、热度等一律用文字
- 除表格与要点外，每个 section 须包含**至少一段连贯的段落式深度解析**

## 开篇结论

本段为结论先行的高洞察摘要，必填。生成时须利用 Phase 3.0 的运营阶段与归因结论。

**格式：** 一段连贯大段落（8–15 句或约 200–400 字）。结论先行：先给出本报告核心结论、本期整体态势、口碑与版本节奏关系、最值得关注的机会与风险，再以简要依据支撑。高洞察：回答本质问题、主要原因、若不干预如何演变；禁止仅复述数据或表格。**本段仅文字，不包含表格或列表。**
