# Game Guide QA - 知识库模板

## 目录结构

```
knowledge_base/
├── items/              # 物品/资源
├── systems/            # 系统机制
├── quests/             # 任务
└── locations/          # 地点
```

## 文件模板

### 物品类 (items/*.md)

```markdown
# [物品名称]

## 基础信息
- 解锁等级: [数字]
- 类别: [作物/加工品/装饰/其他]
- 售价: [数字] [货币单位]

## 获取方式
1. **[方式1]**: [描述]
2. **[方式2]**: [描述]

## 用途
- [用途1]
- [用途2]

## 相关任务
- [任务名称]: 需要[数量]个

## 常见问题
Q: [问题]
A: [答案]
```

### 系统类 (systems/*.md)

```markdown
# [系统名称]

## 概述
[系统简介]

## 解锁条件
- 等级: [数字]
- 前置: [如有]

## 核心机制
1. **[机制1]**: [描述]
2. **[机制2]**: [描述]

## 相关建筑/功能
- [建筑1]
- [建筑2]

## 常见问题
Q: [问题]
A: [答案]
```

### 任务类 (quests/*.md)

```markdown
# [任务名称]

## 任务信息
- 类型: [主线/日常/成就]
- 解锁等级: [数字]
- 前置任务: [如有]

## 任务目标
1. [目标1]
2. [目标2]

## 任务奖励
- [奖励1]
- [奖励2]

## 攻略提示
[完成建议]
```

### 地点类 (locations/*.md)

```markdown
# [地点名称]

## 基本信息
- 解锁等级: [数字]
- 功能: [主要用途]

## 可交互内容
- [内容1]
- [内容2]

## NPC/功能
- [NPC1]: [功能]
- [NPC2]: [功能]

## 相关任务
- [任务1]
- [任务2]
```

## 快速开始

1. 根据游戏内容创建对应的 md 文件
2. 按照模板填写信息
3. 确保关键词与玩家常用说法一致
4. 定期更新，保持与游戏版本同步
